## What's inside?

This Turborepo includes the following:

### Apps and Packages

- `web`: a [vite](https://vitejs.dev/) app
- `api`: an [Express](https://expressjs.com/) server
- `@repo/ui`: a React component library
- `@repo/db`: a PostgreSQL database (Prisma)
